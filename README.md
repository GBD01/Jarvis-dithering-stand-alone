# Jarvis-dithering-stand-alone
Transparency friendly Python script for applying Jarvis dithering to image files.
#	jarvis_dither.py is a Python script to apply Jarvis dithering to images while retaining any transparencies.
#	This script targets python 3.12 or higher. To check your version enter "python --version" in a command window
	if your version of python is less or not installed, download latest version from python.org
#	Install python dependencies by entering the following in a command window
	"pip install numpy pillow keyboard pywin32 win11toast"

Once you have python 3.12 or higher and the dependencies installed, just drag and drop your image onto
The jarvis_dither.py script and it will automatically apply Jarvis dithering and output it as a new file.
